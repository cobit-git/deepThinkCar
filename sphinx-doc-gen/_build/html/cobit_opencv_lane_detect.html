cobit_opencv_lane_detect module¶

class cobit_opencv_lane_detect.CobitOpencvLaneDetect¶

기반 클래스: object

OpenCV 기반 차선 검츨 클래스

이 클래스는 차량의 카메라와 OpenCV의 color기반 오브젝트 디텍션을 이용해서 차량의 앞 차선의 각도를 검출함.

Example

cv_detector = CobitOpencvLaneDetect()

lanes, img_lane = cv_detector.get_lane(img_org)

angle, img_angle = cv_detector.get_steering_angle(img_lane, lanes)

get_lane(frame)¶

입력된 이미지 파일에서 차선을 발견하고, 차선의 좌표값을 어레이로 얻음.

매개변수

frame -- 카메라에서 전달된 차량 앞, 차선의 이미지 파일

Return lane_lines

차선의 좌표값 np 어레이 데이터

Return final_frame

카메라에서 전달된 차량 앞 이미지

get_steering_angle(img_lane, lane_lines)¶

입력된 이미지 파일과 차선의 좌표값 np 어레이에서 차량의 스티어링 휠 회전 각도를 얻음.

매개변수

  • img_lane -- 카메라에서 전달된 차량 앞, 차선의 이미지 파일

  • lane_lines -- 차선의 좌표값 np 어레이 데이터

Return curr_heading_image

차선을 녹색/적색 직선으로 표시한 차량 앞 이미지

Return curr_heading_angle

차선의 각도, 즉 차량의 스티어링 휠이 회전할 각도
